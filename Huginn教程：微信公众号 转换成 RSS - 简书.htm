说明

  • 生成的RSS，是公众号的 全部文章，不止是最新一篇
  • 支持订阅 多个公众号，不止是一个

准备工作

  • 1台Huginn
  • 1枚 phantomjscloud 的API Key（申请地址）

操作步骤

不说制作过程啦，纵有时间写，你也不一定有兴趣读。请 点击下载 Scenario

下载后的操作：

  1. 在 Huginn 的 Credentials 页面，新建一个值，名称是 phantomjs_cloud，值是你申请到的 phantomjscloud 的 API Key


    设置 Credential

  2. 导入下载的 Scenario JSON 文件


    导入 Scenario


    导入后，如图所示，有五个 Agent：


    Huginn 的 Scenario 图示

  3. 编辑 微信公众号 #1 搜索账号，删除我设置的几个公众号。填写进去，你感兴趣的公众号URL


    编辑订阅列表


    query= 后面跟的是，公众号的账号：


    公众号账号


    注：_comment 字段是注释，全部删除也可以，不影响程序运行，随你

  4. 完成！（若有其它需求，如抓取更频繁、过滤文章等，可酌情调整配置）
